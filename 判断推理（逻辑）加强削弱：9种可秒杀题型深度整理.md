# 判断推理（逻辑）加强削弱：9种可秒杀题型深度整理





## 第一部分：九种核心题型详解





### 题型一：双观点型



**逻辑模型**

- **原观点**：$A \rightarrow B$（认为A导致B）
- **反对者**：$C \rightarrow B$（认为C才是B的原因，通常隐含A与B无关或A不是根本原因）
- **削弱反对者**：$A \rightarrow C$

**秒杀逻辑**

- **削弱反对者**：建立A与C的联系，即表明是因为A导致了C，C再导致了B。此时A是更根本的原因，从而削弱了反对者“C才是原因”的独立性。

**经典案例**

1. **哮喘病成因**：
   - 原观点：情绪问题（A）$\rightarrow$ 哮喘（B）。
   - 反对者：神经兴奋性改变（C）$\rightarrow$ 哮喘（B），与情绪无关。
   - 削弱反对者：消极情绪（A）会引起神经兴奋性改变（C）。
2. **皮肤年轻**：
   - 原观点：胶原蛋白（A）$\rightarrow$ 年轻（B）。
   - 反对者：表皮干细胞（C）$\rightarrow$ 年轻（B）。
   - 削弱反对者：胶原蛋白（A）对表皮干细胞（C）的更新至关重要。
3. **脱发成因**：
   - 原观点：营养不均衡（A）$\rightarrow$ 脱发（B）。
   - 反对者：毛囊受损（C）$\rightarrow$ 脱发（B）。
   - 削弱反对者：毛囊受损（C）是营养不均衡（A）导致的。

------



### 题型二：因果倒置型（或引入共同原因型）



**逻辑模型**

- **题干特征**：A与B共存/相关，结论推断 $A \rightarrow B$。
- **削弱方式**：
  1. **因果倒置（最强）**：$B \rightarrow A$（B导致A）。
  2. **另有共因（较强）**：$C \rightarrow A$ 且 $C \rightarrow B$（C同时导致了A和B）。

**秒杀逻辑**

- **公考首选**：直接反过来说，即“因果倒置”。
- **备选（管理联考常见）**：引入共同原因C。如果选项中没有因果倒置，选共同原因。

**经典案例**

1. **睡眠质量**：
   - 题干：关门窗的人（A）睡眠质量差（B） $\rightarrow$ 关门窗导致睡眠差。
   - 削弱（倒置）：睡眠质量差的人（B）更喜欢关门窗睡觉（A）。
2. **徒步与健康**：
   - 题干：远距离徒步的人（A）身体健康（B） $\rightarrow$ 徒步使人健康。
   - 削弱（倒置）：身体健康的人（B）才去徒步（A）。
3. **烟囱与长内衣**（共因示例）：
   - 题干：烟囱起火（A）与买长内衣（B）频率同时高 $\rightarrow$ 烟囱起火导致买内衣。
   - 削弱（共因）：天气寒冷干燥（C）导致了烟囱起火（A）和买长内衣（B）。

------



### 题型三：前提性——搭桥



**逻辑模型**

- **题干特征**：论据提到概念A和B，结论提到概念A和C（或跳跃到新概念）。存在概念跳跃。
- **加强方式**：在跳跃的概念之间建立联系（搭桥）。

**秒杀逻辑**

- 直接在论据的落脚点（B）和结论的落脚点（C）之间建立正向联系。

**经典案例**

1. **杂草稻抗旱**：
   - 论据：基因PAPH1（A）与离子流速（B）相关。
   - 结论：基因PAPH1（A）影响抗旱性（C）。
   - 搭桥：离子流速（B）与抗旱性（C）正相关。
2. **渐冻症（ALS）**：
   - 论据：AM菌（A）产生烟酰胺（B）。
   - 结论：AM菌（A）影响ALS病情（C）。
   - 搭桥：补充烟酰胺（B）能减轻ALS症状（C）（建立烟酰胺与病的联系）。

------



### 题型四：数据比例型（稀释思路）



**逻辑模型**

- **题干特征**：因为有 X% 的 A 都是 B $\rightarrow$ 推出 B 导致 A（或 B 是 A 的原因）。
- **核心概念**：
  - **实际值**：材料中给出的比例 X%。
  - **理论值**：B 在总群体中的自然占比（即如果不考虑A因素，正常人是B的概率）。

**秒杀逻辑（方法总结）**

| **目的** | **判定公式**            | **逻辑解释**                                                 |
| -------- | ----------------------- | ------------------------------------------------------------ |
| **削弱** | **理论值 $\ge$ 实际值** | 如果总人群中B的比例（理论值）本身就很高（大于等于X%），说明在A群体中B的比例并没有异常升高，甚至可能降低了，因此B不是导致A的原因。 |
| **加强** | **理论值 $<$ 实际值**   | 如果总人群中B的比例很低，而A群体中B的比例（X%）显著高于正常水平，说明B与A强相关。 |

**经典案例**

1. **吸烟与肺癌**：
   - 题干：肺癌患者（A）中有70%吸烟（B） $\rightarrow$ 吸烟导致肺癌。
   - 削弱：所有人群（理论值）中吸烟的比例超过70%。
2. **游戏与挂科**：
   - 题干：不及格学生（A）中90%打游戏（B） $\rightarrow$ 打游戏导致不及格。
   - 削弱：打游戏的学生占**全体学生**（理论值）的90%以上（说明不及格群体中打游戏比例并未异常偏高）。
3. **手机与脑癌**：
   - 题干：脑癌患者中68%使用移动电话。
   - 削弱：使用移动电话的人占总人口的比例大于等于68%。

------



### 题型五：对比实验加强



**逻辑模型**

- **题干特征**：将对象分为两组进行实验（有无某种因素），观察结果。
- **加强方式**：
  1. **起点相同**：实验前两组对象的情况一致。
  2. **变量唯一**：排除其他干扰因素（除了实验因素外，其他条件都一样）。
- **削弱方式**：起点不同、存在其他变量（另有他因）。

**经典案例**

- **喝水减肥**：
  - 实验：一组餐前喝水，一组不喝，喝水的瘦得多 $\rightarrow$ 喝水能减肥。
  - 加强：除了餐前喝水，两组的其他情况（如运动量、饮食控制）都是一样的；或实验前两组体重基数一致。

------



### 题型六：共变法局限性（“若没有”假设）



**逻辑模型**

- **题干特征**：采取了措施A，但结果看来没有变化或无效。
- **考察方向**：通过假设“如果没有措施A”会怎样，来判断A是否有效。

**秒杀逻辑**

| **目的**             | **逻辑表述**                  | **含义**                                                  |
| -------------------- | ----------------------------- | --------------------------------------------------------- |
| **加强（证明有效）** | **若没有A，状态会比现在更差** | 说明A起到了遏制恶化的作用（虽然表面没好转，但实际有效）。 |
| **削弱（证明无效）** | **若没有A，状态不会比现在差** | 说明A的存在与否对结果无影响，确实无效。                   |

**经典案例**

1. **祛斑霜**：
   - 现象：用了祛斑霜，斑没减少 $\rightarrow$ 结论：无效。
   - 削弱（证明其实有效）：如果不使用，脸上的斑会更多。
2. **抗癌药**：
   - 现象：吃药后癌症没治愈 $\rightarrow$ 结论：药没用。
   - 削弱（证明其实有效）：如果不服药，症状会比现在严重得多。

------



### 题型七：类比推理型



**逻辑模型**

- **题干特征**：对象A和对象B相似，A有属性X $\rightarrow$ 推测B也有属性X。
- **核心**：类比的有效性取决于“本质”是否相同。

**秒杀逻辑**

- **削弱**：指出A和B在**本质上**存在不同（不能类比）。
- **加强**：指出A和B在**本质上**是相同的（可以类比）。
- **注意**：普通的表面相似（如指甲一样长）是无效类比，必须扣住核心机理。

**经典案例**

1. **树鼩与人类（脑缺血药物）**：
   - 削弱：人脑的构造比树鼩脑更为复杂（本质构造不同，药物可能不通用）。
2. **木星与卫星（含水）**：
   - 加强：卫星和它环绕的行星是同一时期由相同的气体和灰尘形成的（成因本质相同，构成物质可能相同）。

------



### 题型八：枚举归纳型（部分推整体）



**逻辑模型**

- **题干特征**：根据部分样本的调查结果 $\rightarrow$ 推测整体的情况。
- **削弱方式**：
  1. **样本不具代表性**（最常考）。
  2. 样本数量不足。
  3. 存在反例（有特例）。
- **加强方式**：样本具有代表性、数量充足、无反例。

**经典案例**

1. **创新社会调查**：
   - 调查显示大家理解创新社会 $\rightarrow$ 结论：当地人民都理解。
   - 削弱：被调查者主要来自知识产权局（样本特殊，不具代表性，专业人士自然理解）。
2. **全球运动量**：
   - 根据美国和中国运动量下降 $\rightarrow$ 结论：全球运动量缺乏。
   - 加强：其他国家情况与中美大致相同（证明样本有代表性）。

------



### 题型九：推出关系矛盾型



**逻辑模型**

- **题干特征**：结论是条件关系，如“只有A才B”（$B \rightarrow A$）或“如果A就B”（$A \rightarrow B$）。
- **削弱方式**：找矛盾命题（即：满足前件，但不满足后件）。
  - 对于 $A \rightarrow B$，削弱选 **A且非B**。
  - 对于“只有A才B”（即 $B \rightarrow A$），削弱选 **B且非A**（有结果B，但没有条件A）。

**经典案例**

1. **制造业与稳定**：
   - 结论：只有具备强大制造业（A），经济才有高稳定性（B）。逻辑：$B \rightarrow A$。
   - 削弱（B且非A）：有些国家保持经济稳定（B），却没有发达制造业（非A）。
2. **广告与受欢迎**：
   - 结论：只有投入大量广告（A），才有好市场接受度（B）。逻辑：$B \rightarrow A$。
   - 削弱（B且非A）：某产品没有做广告（非A），仍然受到市场欢迎（B）。

## 第二部分：作者实战经验与避坑指南



此部分提炼自讲师的口语讲解和经验总结，**非常关键**。



### 1. 题型判断与解题心态



- **秒杀的含义**：讲义中归纳的这9种答案，在考试中如果出现，通常就是**最强**的加强或削弱项，甚至是**唯一**的正确方向。
- **不要过度预判**：见到这类题型，直接找对应的特征选项，不需要在脑海中构建过于复杂的逻辑链条。
- **关注逻辑链**：做逻辑题的核心是关注“论证过程”（前提 $\rightarrow$ 结论），而不仅仅是看前提或结论。



### 2. 关于“因果倒置”的特殊经验



- **公考 vs. 联考**：
  - **公考（国考/省考）**：主要考“**B导致A**”（直接倒置）。
  - **管理类联考**：可能会考“**C导致A和B**”（共同原因）。
  - 在公考中，如果出现“因果倒置”选项，**闭眼选**，它是最强的削弱。
- **“只有...才...”的时间陷阱**：在某些因果论证中，“只有A才B”不仅仅是逻辑推导，有时代表**时间先后**。即A发生在B之前。如果选项指出B发生在A之前（例如：先有兴趣B，才有长时间学习A），这也是一种因果倒置（时间顺序反了）。



### 3. 关于“数据比例型”的避坑



- **混淆概念**：千万不要选那些谈论“增长率”的选项。
  - 例如：题干讨论“占总人数的比例”（绝对占比），干扰项会说“比去年增加了68%”（增长率）。**比例 $\neq$ 增长率**。
  - **错误选项示例**：选项C说“人数增加了68%”，这是典型的干扰项，与“占总人口68%”是两码事。
- **基准值理解**：必须理解“理论值”就是**总人数/全人群**中的比例。削弱的核心是证明“特殊群体的数据”即是“正常数据”，说明两者没有相关性。



### 4. 关于“类比推理”的原则



- **慎选类比**：一般情况下，类比选项是弱选项，不选。
- **何时选类比**：只有当选项明确指出了**本质相同**（加强）或者**本质不同**（削弱）时，类比选项才是有效的秒杀项。比如涉及到“构造原理”、“形成机制”等本质层面。



### 5. 关于“考情分析”与复习建议



- **重点关注一拖五**：对于地市级和行政执法类试卷，逻辑部分要重点复习“一拖五”题型（综合判断推理），这是拿分关键。
- **考点分布**：副省级试卷通常加强削弱题较多；地市级和执法类试卷虽然也有，但分析推理（组合排列）占比大。
- **无关项**：做题时除了找加强削弱，还要学会快速识别“无关项”（讲师提及有专门视频讲解），这有助于排除干扰。

## 第三部分：九种题型对比速查表



| **题型**     | **题干核心逻辑**                             | **削弱秒杀方向**                                            | **加强秒杀方向**         | **备注**           |
| ------------ | -------------------------------------------- | ----------------------------------------------------------- | ------------------------ | ------------------ |
| **双观点**   | 原: $A \rightarrow B$; 反: $C \rightarrow B$ | **$A \rightarrow C$** (A导致C)                              | -                        | A是根本原因        |
| **因果倒置** | 现象A与B共存 $\rightarrow$ $A \rightarrow B$ | **$B \rightarrow A$** (最强) **$C \rightarrow A+B$** (共因) | -                        | 公考见倒置直接选   |
| **前提搭桥** | 概念A $\rightarrow$ 概念B (跳跃)             | -                                                           | **建立A与B的联系**       | 缺桥补桥           |
| **数据比例** | A中有X%是B $\rightarrow$ B导致A              | **总人群中B比例 $\ge$ X%**                                  | **总人群中B比例 $<$ X%** | 比较理论值与实际值 |
| **对比实验** | 两组实验对比                                 | **起点不同 / 另有他因**                                     | **起点相同 / 变量唯一**  | 排除干扰变量       |
| **共变法**   | 措施A看似无效                                | **没A不会更差** (真无效)                                    | **没A会更差** (其实有效) | 假设反事实         |
| **类比推理** | A像B $\rightarrow$ B也...                    | **本质不同**                                                | **本质相同**             | 必须扣住“本质”     |
| **枚举归纳** | 部分 $\rightarrow$ 整体                      | **样本不具代表性** / 数量少                                 | **样本具代表性**         | 调查类题目常用     |
| **推出矛盾** | $A \rightarrow B$                            | **A且非B**                                                  | -                        | 找反例             |
