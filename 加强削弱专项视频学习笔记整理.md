# 加强削弱专项视频学习笔记整理





## 一、 核心做题流程与方法体系



本部分涵盖从审题到作答的标准五步法，以及核心的力度比较原则。



### 1. 五步做题法



**第一步：审题定方向**

- **操作技巧**：
  1. **看提问**：先读题干末尾，明确题型（加强、削弱、不能加强、不能削弱）。
  2. **找论点**：根据标志词（因此、所以、结论是、宣称、认为）定位核心论点。
  3. **找论据**：向前梳理支撑论点的材料（数据、实验、现象、理论）。
- **示例**：若提问“不能加强”，需排除3个加强项，选削弱或无关项。

**第二步：梳理论证结构**

- **操作技巧**：
  1. **判断论点类型**：
     - **因果类**：A导致B。
     - **非因果类**：B成立/不成立（如判断某效果是否存在）。
  2. **分析漏洞**：找出论据与论点间的逻辑断层（如相关性≠因果性、样本偏差、偷换概念）。
  3. **明确方向**：
     - 加强方向：排除他因、搭桥、解释原理。
     - 削弱方向：因果倒置、引入他因、否定论点、拆桥。

**第三步：选项逐一分析**

- **判断标准**：所有分析必须围绕“论点核心”。
- **选项分类**：
  - **有效选项**：加强项（搭桥、排除他因等）、削弱项（否定论点、因果倒置等）。
  - **无效选项**：无关项（话题偏离）、不明确项（模糊表述）、主观/权威项（无客观依据）。

**第四步：力度比较**

- **适用场景**：当存在多个有效选项时，必须依据逻辑关联性进行排序。

**第五步：检查复核**

- **检查重点**：是否忽略“最”字；是否误选不明确项；是否混淆论点核心（如将副作用误判为无效）。

### 2. 选项力度排序规则表



作者明确强调了不同逻辑手段的力度层级，需严格遵循。

| **题型**   | **力度排序（由强到弱）**             | **核心逻辑说明**                             |
| ---------- | ------------------------------------ | -------------------------------------------- |
| **加强题** | **1. 搭桥** (直接建立因果/逻辑链路)  | 解释原理，直接修补逻辑漏洞，关联性最强。     |
|            | **2. 排除他因** (针对因果类)         | 排除了其他可能性，纯化因果关系。             |
|            | **3. 加强论点** (直接肯定结论)       | 直接重申结论的正确性。                       |
|            | **4. 加强论据** (补充数据/实验/样本) | 仅增加证据数量或可靠性，未解决核心逻辑漏洞。 |
|            | **5. 类比加强**                      | 通过类似事物佐证，力度较弱。                 |
| **削弱题** | **1. 否定论点 = 因果倒置**           | 直接证明结论是错误的，或因果关系完全相反。   |
|            | **2. 拆桥** (切断论据与论点关联)     | 说明论据无法推导出论点。                     |
|            | **3. 他因削弱** (针对因果类)         | 提出其他原因导致结果，但不完全排除原原因。   |
|            | **4. 削弱论据** (否定数据/样本偏差)  | 攻击证据的可靠性，未直接否定结论本身。       |
|            | **5. 类比削弱**                      | 通过类似事物反证，力度较弱。                 |

## 二、 典型习题思维还原与深度解析



本部分还原视频中两道典型例题的完整思维链条，展示如何应用上述理论。



### 案例一：因果类加强题



**1. 题干核心信息**

- **论据**：调查1000人5年，每周运动>3次的人心血管病发病率比运动<1次的人低40%（呈现相关性数据）。
- **论点**：规律运动（每周>3次）能降低心血管疾病的发病风险（A导致B）。
- **提问**：最能**支持**上述结论。

**2. 论证分析与漏洞**

- **逻辑关系**：因果类论点（运动 -> 降风险）。
- **核心漏洞**：**相关性不等于因果性**。
  - 可能因果倒置（身体好才有精力运动）。
  - 可能存在他因（运动的人饮食更健康）。
  - 可能是偶然相关。
- **加强方向**：解释原理（搭桥）、排除他因、排除因果倒置。

**3. 选项深度分析表**

| **选项内容**                                               | **选项分析与定性**                                           | **判决**       |
| ---------------------------------------------------------- | ------------------------------------------------------------ | -------------- |
| **A** 另500人追踪3年，结果一致（运动多发病率低）。         | 补充了新样本和时间，结果与原论据一致。属于**加强论据**（增加数据量），但未解决相关性≠因果性的漏洞。 | 有效（弱）     |
| **B** 运动促进循环->减少脂肪->降低动脉硬化->减少心血管病。 | 从原理层面解释了“运动”如何导致“低风险”，建立了具体因果链路。属于**搭桥加强**。 | **有效（强）** |
| **C** 运动>3次的人同时也定期体检。                         | 指出“定期体检”这一第三方因素可能导致发病率低。属于**他因削弱**。 | 无效（排除）   |
| **D** 发病风险与遗传有关，有优势者不运动也不易病。         | 讨论“遗传”对风险的影响，未提及“运动”的作用。属于**无关选项**。 | 无效（排除）   |
| **E** 运动1-2次与>3次发病率差异不大。                      | 未强化“运动导致低风险”的逻辑，且可能暗示运动效果饱和。属于**不明确选项**。 | 无效（排除）   |

**4. 最终决策**

- **力度比较**：选项B（搭桥，直接解决核心逻辑漏洞） > 选项A（加强论据，仅增加证据量）。
- **答案**：B

### 案例二：非因果类削弱题（否定论点）



**1. 题干核心信息**

- **宣称**：连续服用1个月，减重5公斤以上。
- **论据**：200名超标志愿者，连续服用1个月，平均减重6公斤。
- **论点**：该减肥药确实能有效减重（即效果存在）。
- **提问**：最能**质疑**上述结论。

**2. 论证分析与漏洞**

- **逻辑关系**：非因果类论点（验证效果）。
- **核心漏洞**：**实验未设置对照组**。无法排除减重是由志愿者参与实验后的心理作用、饮食控制等其他因素导致的。
- **削弱方向**：直接否定论点（证明无效）、引入对照组证明无效、指出样本严重偏差。

**3. 选项深度分析表**

| **选项内容**                                                 | **选项分析与定性**                                           | **判决**       |
| ------------------------------------------------------------ | ------------------------------------------------------------ | -------------- |
| **A** 志愿者每天减少2000卡摄入并增加1小时运动。              | 指出饮食和运动可能是减重原因。属于**他因削弱**，但未直接证明药无效。 | 有效（中）     |
| **B** 主要成分是膳食纤维，仅增加饱腹感减少摄入。             | 虽无法燃烧脂肪，但“增加饱腹感导致少吃”也是减重路径，未否定“有效减重”。属于**不明确选项**。 | 无效（排除）   |
| **C** 150人基数超100公斤，易因脱水大幅减重。                 | 指出样本特殊（基数大），减重可能是脱水。属于**削弱论据**（样本偏差），未直接否定药效。 | 有效（中）     |
| **D** 另设安慰剂组，平均减重5.5公斤（与服药组6公斤差异极小）。 | 通过对照组证明，不吃有效成分也能减重，说明减重是安慰剂效应，药物本身无效。属于**否定论点**。 | **有效（强）** |
| **E** 部分志愿者出现腹泻、头晕副作用。                       | 讨论的是“副作用”，论点讨论的是“减重效果”。属于**无关选项**。 | 无效（排除）   |

**4. 最终决策**

- **力度比较**：选项D（否定论点，直接证明无效） > 选项A（他因削弱）/ 选项C（削弱论据）。
- **答案**：D

## 三、 作者实战经验与应试心得提取



本部分专门提取视频作者在讲解过程中强调的经验、技巧及避坑指南。



### 1. 易错与避坑提醒



- **关于不明确选项**：凡是出现“可能、或许、部分、有些、尚未证实、有待研究”等模糊表述的，通常是无效选项。不要把“未验证”当成“无效”。
- **关于“最”字**：必须时刻注意提问中的“最能支持/削弱”。不要看到一个有效项就选，必须进行力度比较。
- **关于副作用**：如果论点核心是讨论“效果/功能”，选项中提及“副作用/成本”通常属于无关项（除非论点综合考量性价比）。
- **关于提问方式**：遇到“不能加强/削弱”的题目，逻辑是排除有效项，选“无关项”或“反向项”（通常选无关项居多，需具体分析），而不是单纯找反义词。
- **关于因果类无关项辨析**：他因削弱的“他因”必须能解释结果。如果“他因”与结果无逻辑关联（如“穿浅色衣服”与“减重”），则纯属无关，不能强行解释。



### 2. 实战解题技巧



- **审题节奏**：如果题干背景信息过长（如冗长的实验细节描述），可以跳读，直接抓“论点标志词”和“核心数据/结论”，避免被干扰信息消耗注意力。
- **优先排除法**：做题第一步应快速排除明显的话题偏离项、主观感受项，减少大脑负荷。
- **因果类秒选特征**：
  - 若论点是因果类，选项中出现“因果倒置”、“他因”、“搭桥”的，通常就是正确答案的候选，需优先关注。
  - 因果类题目中，“解释原理（搭桥）”和“排除他因”的力度，通常天然强于“补充相关性数据（加强论据）”。
- **力度判断原则**：不需要死记硬背所有规则，核心心法是看**逻辑关联性**。选项与论点核心逻辑结合得越紧密，力度就越强（如搭桥是直接修路，补充论据只是路边种树）。
- **削弱题最高力度**：直接否定论点（证明结论错误）是削弱题中的最高级。如果选项能做到这一点，基本就是必选项。



### 3. 核心思维总结



- **论点绝对核心化**：做题的全过程（分析、判断、比较）都必须死死咬住“论点”。选项分析稍微偏离论点一步，就可能是陷阱。
- **漏洞导向**：解题的本质是找茬。加强是补窟窿，削弱是捅窟窿。看不出漏洞，就很难精准定位加强/削弱的方向。
- **客观力度**：力度比较要依据客观逻辑规则（如搭桥>加强论据），不能凭借个人语感或主观觉得哪个更有道理。
