# 花生老师加强削弱专项做题方法完整复盘





## 一、 核心题型分类概览



加强削弱题型主要分为三大板块：

1. **削弱方法——归因类**（侧重因果分析）
2. **削弱方法——一般质疑类**（侧重逻辑推导）
3. **加强方法——三大类总结**（支持、前提、解释）

------



## 二、 第一部分：削弱方法——归因类





### 1. 核心识别特征



- **判断标准**：题干由“既定事实 + 因果分析”构成，重点在于解释“为什么”。
- **识别要点**：
  - 能识别出“原因”和“结果”。
  - 题干在解释现象背后的原因（A导致B）。
- **示例**：事实（现在爱学习）+ 原因（因为考公）。



### 2. 归因类的六大子类型



| **类型名称**        | **识别特征**                                              | **核心逻辑**                                                 | **典型示例**                                         | **重要程度** |
| ------------------- | --------------------------------------------------------- | ------------------------------------------------------------ | ---------------------------------------------------- | ------------ |
| **1. 比较实验归因** | 两组实验（实验组+对照组）；同一时间，不同主体；横向对比。 | 变量需正确分组；选项需回归实验组结果。                       | 喂酒精小鼠 vs 不喂酒精小鼠；结论：酒精影响恐惧记忆。 | ★★★★★        |
| **2. 时间对比归因** | 同一对象；过去 vs 现在；纵向对比。                        | **用变化解释变化**；变化是导致结果的原因。                   | 过去不爱学习 vs 现在爱学习；原因：考公。             | ★★★★★        |
| **3. 一般归因**     | 无比较实验，无时间对比；针对具体事实的因果分析。          | 单独分析某现象原因。                                         | 分析某具体事件发生的原因。                           | ★★★★         |
| **4. 直接根因**     | 原观点 vs 对立观点；对同一结果的不同解释。                | **质疑对方**：原观点→对方观点→结果（对方是表象）。 **支持对方**：对方观点→原观点→结果（原观点是表象）。 | A导致C vs B导致C。 质疑B：A导致B导致C。              | ★★★          |
| **5. 构成对比实验** | 只有一组实验（缺对照组）；论证不完整。                    | 选项需“补充对照组”；题干纵向则补横向，题干单一则补对比。     | 题干只说了实验组情况，选项补充对照组情况。           | ★★★          |
| **6. 补充对比实验** | 选项本身是一个完整的对比实验。                            | 选项独立包含“实验组+对照组+结果”。                           | 选项直接给出两组对比数据来支持/削弱。                | ★★★          |



### 3. 归因类的三大核心削弱方法



| **方法**        | **核心逻辑**                                             | **典型特征/操作**                                            | **力度**             |
| --------------- | -------------------------------------------------------- | ------------------------------------------------------------ | -------------------- |
| **1. 否定此因** | 不探讨其他原因，**直接否定**题干提出的原因。             | 题干：A导致B。 削弱：A对B没有作用。                          | **最强** (≈因果倒置) |
| **2. 因果倒置** | 在相对确定条件下，将因果关系颠倒。                       | 题干：A导致B。 削弱：其实是B导致A。 前提：时间顺序错误。     | **极强** (≈否定此因) |
| **3. 他因削弱** | 引入B、C、D等其他因素，降低A是原因的可能性（“搅浑水”）。 | **回归实验**：因素须影响实验结果。 **分组正确**：因素须在同一实验组内（如实验组除了吃药还运动）。 | 较弱                 |



### 4. 归因类的两大加强方法



| **方法**        | **核心逻辑**                                                 | **操作方法**                                         |
| --------------- | ------------------------------------------------------------ | ---------------------------------------------------- |
| **1. 排除他因** | 排除其他干扰因素，使题干原因成为唯一可能，实现**控制变量**。 | 选项说明其他可能影响结果的因素“不存在”或“已被排除”。 |
| **2. 解释说明** | 对因果关系进行补充阐释，让人“恍然大悟”。                     | 提供支撑点，使因果链条更清晰。                       |



### 5. 特殊概念与技巧





#### 虚假关联 vs 他因削弱



- **虚假关联 (C→A, C→B)**：指出A和B平行，无因果关系，都是由共同因素C导致的。**力度强于一般他因**。
  - *例子*：夏天(C)导致冰淇淋销量增(A)和溺水事故增(B)，A推不出B。
- **他因削弱**：承认A可能是原因，但B、C也可能是。



#### “三圈质疑”与“四圈支持”记忆法



- **三圈质疑（削弱）**：
  - **同因异果**：原因相同，结果不同（我俩都吃食堂，我中毒你没中毒）。
  - **异因同果**：原因不同，结果相同（我吃食堂你没吃，我俩都中毒）。
- **四圈支持（加强）**：
  - **异因异果**：原因不同，结果不同（我吃食堂中毒，你没吃没中毒）。



### 6. 归因类常见错误选项特征



- **虚假他因**：提出的他因**没有回归到实验结果**中，最具迷惑性。
- **“有些”**：数量不明确，力度极弱（除非用于质疑“所有否定”）。
- **实验缺陷**：质疑实验设计或样本，而非因果关系，通常不选。
- **不提原因**：完全不涉及原因的描述是无关选项。

## 三、 第二部分：削弱方法——一般质疑类





### 1. 核心特征



- **定义**：除归因类之外的所有削弱题型（不讨论因果，只讨论论据推结论）。
- **三大正确选项特征**：
  1. **主体一致**：讨论对象A必须对应A。
  2. **话题一致**：讨论话题X必须围绕X。
  3. **符合题干**：一般不选“有些”。



### 2. 一般质疑的四大子类型





#### 类型1：有论据有结论（最常见）



- **结构**：论据1+论据2... → 结论。
- **三大质疑切入点**：
  1. **质疑论证过程（95%情况）**：
     - **漏洞A：考虑不周全（以偏概全）**。方法：补充反例。
     - **漏洞B：存在断点（缺少桥梁）**。方法：拆桥，指出论据推不出结论。
     - **造句技巧**：“虽然论据是这样，但是你没有考虑[选项内容]，所以结论不成立。”
  2. **质疑结论（中等频率）**：
     - 特征：结论出现论据中没有的**新内容**（无中生有）。
     - 逻辑：A 且 非B（论据成立，但结论不成立）。
     - *注意*：选项必须包含结论中的“新内容”。
  3. **质疑论据本身（5%情况）**：
     - 仅当论据是“个人观点/判断”时可质疑；若是“客观事实/领导发言”一般不质疑。



#### 类型2：有结论无论据



- **结构**：背景/分析 → 结论（无直接论据支撑）。
- **质疑方法**：找A与非A的矛盾。
  1. 直接否定结论（他说A，你说非A）。
  2. 得出相反结论。
  3. 合理说明不成立。



#### 类型3：严谨逻辑关系



- **识别**：出现逻辑关联词（如果...那么，只有...才，只要...就，除非...否则）。
- **质疑方法（找矛盾）**：
  - **如果A→B**：矛盾是 **A 且 非B**（A成立但B不成立）。
  - **只有A才B (B→A)**：矛盾是 **B 且 非A**（B成立但没有A）。
  - **A是B的必要条件 (B→A)**：矛盾是 **B 且 非A**。



#### 类型4：比例论证



- **识别**：题干只分析了**分子**（数量），用数量代替比例。
- **核心漏洞**：忽略分母影响。
- **质疑方法**：**直接找包含分母数据的选项**。
  - 投诉电话多 ≠ 质量差（分母：市场占有率）。
  - 事故数量多 ≠ 不安全（分母：总运营量/总车辆数）。
  - 迷路司机多 ≠ 导航无效（分母：司机总数）。

## 四、 第三部分：加强方法——三大类总结



**重要认知**：支持论证、前提假设、解释说明本质逻辑一致，不需要过度拆分，统称为“加强”。



### 第一大类：支持论证类



- **识别**：提问含“支持”、“加强”。

| **方法**        | **核心逻辑**                         | **识别与操作**                                           | **重要度**     |
| --------------- | ------------------------------------ | -------------------------------------------------------- | -------------- |
| **1. 搭桥法**   | **结论出现新内容**，形成断点。       | 选项同时包含**论据关键信息 + 结论新内容**，建立联系。    | ★★★★★ (最常考) |
| **2. 解释说明** | **用变化解释变化**。                 | 解释为什么会出现该结果，给人“恍然大悟”感。               | ★★★★★          |
| **3. 必要条件** | 虽然不能100%证明，但**缺了它不行**。 | **否定代入法**：假设选项不成立，推理崩塌，则为必要条件。 | ★★★★           |
| **4. 补充论据** | 补充事实或数据，增强说服力。         | 填补论证缺陷，或在论证完整时增加支持证据。               | ★★★★           |
| **5. 举例支持** | 用具体例子支持（如“有些”）。         | 力度极弱，通常不选，仅在无更好选项时考虑。               | ★              |

- **注意事项**：
  - **类比支持极弱**：人 vs 小白鼠，一般不选。
  - **简单重复不如补充论证**：单纯重复论点力度弱。



### 第二大类：前提假设类



- **识别**：提问含“前提”、“假设”。
- **解法**：等同于支持类，主要考查：
  1. **搭桥法**（连接断点）。
  2. **补缺法**（补充漏洞）。
  3. **必要条件**（否定代入验证）。



### 第三大类：解释说明类



- **识别**：提问含“解释这一现象”。
- **解法**：
  - 找矛盾现象（如：价格降质量升但销量降）。
  - 选项需合理解释矛盾，联系实际，让人“恍然大悟”。
  - 若题干涉及对比，选项也应包含对比。

------



## 五、 总结：力度排序与核心逻辑





### 1. 削弱力度排序



**归因类**：

1. **否定此因 ≈ 因果倒置** （最强）
2. 虚假关联
3. 一般他因削弱

**一般质疑类**：

1. **质疑结论**（指出无中生有）/ **直接否定结论**
2. 质疑论证过程（考虑不周全/拆桥）/ 严谨逻辑矛盾
3. 比例论证（找分母）
4. 质疑论据（最弱）



### 2. 加强力度排序



1. **解释说明 ≈ 搭桥法** （最强）
2. 必要条件（否定代入崩溃）
3. 补充论据
4. 举例支持（最弱）

## 六、 作者实战经验与避坑指南 (重点提取)



以下内容整理自作者在讲解过程中的实战心得、易错提醒及个人建议：



### 实战做题技巧



1. **造句法解一般质疑**：遇到有论据有结论的质疑题，直接套用万能句式——“虽然论据是这样，但是你没有考虑【选项内容】，所以结论不成立”。
2. **比例题视角**：做比例论证题时，要关注“我们自己的视角”，**直接找包含分母数据**的选项，分母才是答案，不要被其他无关选项干扰。
3. **搭桥法优先**：在支持类题目中，**搭桥法考查频率高达80%**。只要看到结论出现了论据里没有的新内容，立刻找同时包含“论据关键词”和“结论新内容”的选项。
4. **严谨逻辑优先验证**：在质疑题选项中，如果看到有明显体现严谨逻辑关系（如如果...那么）的选项，**优先验证**它，它很可能是正确答案。
5. **必要条件验证**：使用**否定代入法**。问自己：“如果没有这个选项，整个推理过程还能成立吗？”如果不能，这就是必要条件。
6. **解释题的感觉**：解释说明的正确选项通常读完后会给你一种“恍然大悟”的感觉，“原来如此”是判断标准。



### 易错与避坑提醒



1. **警惕“有些”**：
   - 如果选项中出现“有些”，一般不选，因为数量不明确，力度极弱。
   - **唯一例外**：当题干是“所有A都不是B”（全称否定）时，“有些A是B”可以作为有效质疑。
2. **警惕虚假他因**：这是归因类最常见的错误选项。必须检查提出的他因是否**回归到实验结果**中。如果因素存在但根本不影响实验组结果，就是无关选项。
3. **警惕类比**：无论是支持还是削弱，类比（如题干说人，选项说小白鼠）的力度都**极弱**，一般不选。
4. **警惕简单重复**：支持题中，简单重复题干观点的选项力度弱于补充具体论据或措施的选项。
5. **归因题必须提原因**：归因类题目的核心是找“为什么”，如果选项完全不涉及原因，只是描述现象，肯定是无关选项。
6. **实验缺陷通常不选**：指出实验设计有问题、样本有问题，通常不是归因类的正确答案，因为我们要质疑的是**因果关系**，而不是实验过程。



### 个人建议与心得



1. **不要刻意拆分加强题**：支持、前提、解释这三类，底层逻辑是一样的。前提和解释只是支持的特殊表现形式，不要为了分类而分类，要统一用加强的思维解题。
2. **关注新内容**：在质疑结论题中，如果结论出现了“新内容”（论据未提及），有效质疑**必须**涉及这个新内容。
3. **理解“变化”**：时间对比归因和解释说明类题目，核心逻辑往往是“用变化解释变化”。
4. **心态建设**：“有结论无论据”这类题其实非常直白，正确率通常很高，只要敢于说反话（找A与非A的矛盾）即可，不要害怕。
