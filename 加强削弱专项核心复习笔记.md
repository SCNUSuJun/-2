# 加强削弱专项核心复习笔记





## 第一部分：考情分析与题型定位





### 1. 题量与试卷差异



| **试卷类型**      | **题量特征** | **题型结构**                              |
| ----------------- | ------------ | ----------------------------------------- |
| **地市级/执法卷** | 共5题        | 含“一拖五”题型，加强削弱（含前提型）共5题 |
| **副省级卷**      | 多于地市级   | 无“一拖五”题型                            |



### 2. 前提型题目定位



前提型被定义为国考中的“送分题”，具有以下鲜明特征：

- **必考规律**：除某年地市级卷外，近年几乎每卷必考1道。
- **考察重点**：99%概率考察**“搭桥”**，极少涉及其他前提类型（如“镁离子”类）。
- **解题策略**：优先寻找“搭桥项”，无需考虑其他，以节省时间。

------



## 第二部分：前提型解题方法



核心逻辑是**补全题干论证的逻辑断层**。根据题干是否存在“概念跳跃”选择方法。



### 1. 搭桥法（适用于“概念跳跃”类）



**核心步骤**

1. **定提问**：确认问法为“前提”、“假设”、“必要条件”。
2. **拆论证**：通过标志词（因此、所以、因为）拆分论据（A→B）和论点（A→C）。
3. **找断层**：识别论据与论点中未直接关联的核心概念（断层为B与C）。
4. **选搭桥**：直接寻找连接“断层概念”（B→C）的选项。

**典型例题解析**

| **例题主题**    | **论证结构拆解**                                     | **断层识别**                     | **正确选项逻辑（搭桥）**                     | **干扰项排除**                                               |
| --------------- | ---------------------------------------------------- | -------------------------------- | -------------------------------------------- | ------------------------------------------------------------ |
| **PM2.5与癌症** | 论据：PM2.5→炎症/应激反应 论点：PM2.5→癌症风险       | **炎症/应激反应** 与 **癌症**    | 建立“炎症/应激反应”与“癌症”的关联            | 不看其他无关选项                                             |
| **苦味感知**    | 论据：苦味感知是防御机制 论点：苦味感知→免于有毒物质 | **防御机制** 与 **免于有毒物质** | **大部分有毒物质是苦的**（建立苦与毒的联系） | **同构选项排除**：仅解释苦味感知过程或受体分布（如选项A、C）均未提及有毒物质，直接排除 |
| **植物抗旱**    | 论据：基因→水流速 论点：基因→抗旱性                  | **水流速** 与 **抗旱性**         | **植物水流速与抗旱能力正相关**               | /                                                            |



### 2. 反向验证法（适用于“无概念跳跃”类）



核心逻辑

前提是“必要条件”（没它不行）。否定选项后，若结论必然不成立，则该选项为前提。

**操作步骤**

1. **判断**：论据论点话题一致，无明显跳跃。
2. **验证**：逐一否定选项。
   - 否定后结论失效 → 是前提。
   - 否定后结论仍可能成立 → 排除。

**典型例题解析：听声断酒**

- **题目**：论据与论点均围绕“通过花听声判断酒质”。
- **选项A验证**：
  - 原句：酒液流动的“花听”声与其他酒类不同。
  - 否定后：声音与其他酒类相同。
  - 结果：声音若相同，无法区分好坏，**结论不成立**。A是前提。
- **选项D验证**：
  - 原句：判断好坏需结合“花听”声和酒液外形。
  - 否定后：判断好坏**无需**结合外形（仅听声音也可以）。
  - 结果：结论仍可能成立。D不是前提。

## 第三部分：加强削弱三大核心论证结构





### 1. 因果共存型



**结构特征**：先描述A与B同时发生（共存），结论推测A是B的原因（A→B）。

| **操作类型** | **核心逻辑**                      | **关键要求**                   | **例题：工具使用（A）与脸型变化（B）**                       |
| ------------ | --------------------------------- | ------------------------------ | ------------------------------------------------------------ |
| **削弱**     | **他因削弱**：找与A共存的C，且C→B | 他因C必须与A**时间、主体一致** | **选项B**：两百万年前（时间一致）食物变化（C），且食物变化影响脸型（C→B）。 *排除A*：灵长类动物（主体不一致，不当类比）。 |
| **加强**     | **解释因果**：说明A如何导致B      | 补全A到B的链条                 | /                                                            |

**例题补充：人类活动与反刍动物**

- **削弱**：同一时期（共存），大型食肉动物增多（C）且捕杀反刍动物（C→B）。
- **注意**：若选项只提C→B但未说明C与A“同一时期”，则是无关项。



### 2. 知果求因型



**结构特征**：已知结果B，推测原因是A（B是因为A）。

| **操作类型** | **核心逻辑**                        | **关键要求**           | **例题：小果占比大（B）是因套带藻（A）**                     |
| ------------ | ----------------------------------- | ---------------------- | ------------------------------------------------------------ |
| **削弱**     | **他因削弱**：除A外，其他原因C导致B | C需与A共存             | **选项D**：今年未追加施肥（C），导致果实变小（C→B）。 *排除C*：“未分大小”表述模糊，“不一定”是不明确选项。 |
| **加强**     | **排除他因** / **解释A→B**          | 避免“可能”等不明确表述 | **例题2贝类红霉素**：体内有合成基因（C）且基因表达产生红霉素（C→B），直接指出原因是自身而非微生物。 |



### 3. 知因求果型



**结构特征**：已知原因/条件A，推测会导致结果B（A→B）。

| **操作类型** | **核心逻辑**                   | **关键要求**                         | **例题：投电梯广告（A）利润涨（B）**                         |
| ------------ | ------------------------------ | ------------------------------------ | ------------------------------------------------------------ |
| **削弱**     | **有因无果**：A存在，但B不发生 | 需保证**条件一致**（同主体或同规模） | **选项B**：规模一致的S店（条件一致），投广告后利润下降（有A无B）。 *排除A*：与没投广告的H店比（主体条件不一致，H店可能位置差）。 |
| **加强**     | **解释因果**：说明A如何导致B   | 明确直接关联                         | **例题2牛奶助眠**： 削弱：色氨酸无法穿过血脑屏障（从原理否定A→B）。 加强：色氨酸能转化为血清素促进睡眠。 |

## 第四部分：国考高频模型题（秒杀技巧）





### 1. 根本原因型



模型：原观点 A→B；反对者 C→B。

提问：削弱反对者 / 支持原观点。

| **解题方向**   | **核心套路**                          | **例题：情绪（A）vs 炎症（C）导致哮喘（B）**          |
| -------------- | ------------------------------------- | ----------------------------------------------------- |
| **削弱反对者** | **找 A→C**（说明A导致C，A是根本原因） | **选项D**：消极情绪（A）会提高气道炎症（C）的发生率。 |
| **支持反对者** | **找 C→A**（说明C导致A，C是根本原因） | /                                                     |



### 2. 统计样本型



模型：通过样本数据推导整体结论。

漏洞：样本是否具有代表性（幸存者偏差）。

- **削弱**：样本不具代表性（如样本仅包含特定人群）。
- **例题**：调查显示冰雪旅游需求高。
  - **削弱项**：参与调查的几乎都是已了解冰雪旅游的爱好者（幸存者偏差，无法代表大众）。



### 3. 方式目的型



**模型**：通过方式A，达到目的B。

- **削弱思路**：
  1. 方式A不可行。
  2. 方式A达不到目的B（A实现了，但B没实现）。
- **例题**：互联网升级系统（A）→交互便捷（B）。
  - **削弱项**：接入互联网后稳定性受干扰，无法正常交互（达不到目的）。
  - **排除项**：升级成本高、需培训（仅提难度，未否定目的达成）。

## 第五部分：无关选项与避坑指南





### 1. 无关选项判定标准



只要选项不覆盖论证核心的“A与B”关系，即为无关。

| **无关类型**  | **示例（论证：藤蔓植物A → 减少大火B）** | **判定理由**         |
| ------------- | --------------------------------------- | -------------------- |
| **只提A**     | 藤蔓植物生长速度快                      | 仅描述A特征，未关联B |
| **只提B**     | 大火多由雷击引发                        | 仅描述B特征，未关联A |
| **无关概念C** | 藤蔓植物提供动物栖息地                  | 话题完全脱离A与B     |



### 2. 常见易错点



- **不明确选项一律排除**：包含“可能”、“不一定”、“尚未清楚”的选项，力度极弱。
- **对比实验陷阱**：
  - **错误**：不同条件的主体对比（甲店投了涨，乙店没投跌 → 乙店情况不能说明甲店）。
  - **正确**：同一主体前后对比，或条件完全一致的主体对比。
- **“不能支持”题型**：正确答案90%是**无关项**，而非削弱项。因为削弱项太明显，无关项更具迷惑性。

## 第六部分：作者经验之谈（实战精华）





### 1. 实战做题流程



1. **看提问（10秒）**：明确题型，重点标注“最能”、“不能”。
2. **找论据论点（30秒）**：利用标志词（因此、所以）或首尾句，用字母（A→B）简化核心逻辑。
3. **定结构（20秒）**：判断是因果共存、知果求因、知因求果还是特定模型。
4. **析选项（60秒）**：先排除无关项，再识别有效项类型，最后比力度。
5. **验答案（10秒）**：反问自己选项是否直击A→B，前提型是否补全断层。



### 2. 独家做题心得与技巧



- **同构选项秒杀**：在前提型题目中，如果两个选项结构和意思相近（如都只解释了论据的机制），可以直接同时排除，因为它们都不可能是正确答案。
- **反向验证易错点**：若选项包含“且”关系（需A且B），否定后是“非A**或**非B”。只要其中一个条件不满足结论仍成立，该选项就不是必要前提。
- **他因削弱核心**：在因果共存和知果求因中，**时间/主体一致**是他因削弱的绝对前提。只提“C导致B”但不确定C是否与A共存的，是无关项。
- **知因求果的本质**：削弱必须是“有因无果”（A存在但B没发生），解释因果（A→B）是加强的王道。
- **模型题秒杀**：
  - 根本原因型：看到削弱反对者，直接找 **A→C**。
  - 统计样本型：看到调查推整体，直接找 **样本偏差**。
- **力度比较原则**：必然 ＞ 可能；直接 ＞ 间接；搭桥 ＞ 加强论据。



### 3. 个人建议与心态



- **关于预测**：加强削弱是“严谨模块”，方法固定，不需要过度预测考勤，掌握方法就是拿到分。
- **拿分优先级**：前提型（搭桥）、因果共存（他因）、根本原因型是“送分题”，必须保证正确率。
- **止损策略**：单题限时2分钟。若超过1分钟无思路，直接根据“论证核心”排除无关项，蒙一个最相关A与B的选项。
- **复盘重点**：不要只看对错，要记录“为何错”（结构判断错？无关项没排掉？）。
- **方法迁移**：每做一道题都要总结它属于哪类结构，用了什么方法，做到“做一道会一类”。
- **资源推荐**：重点做近5年国考真题，特别是地市级/执法卷，因为命题更严谨、少争议。
